## 📘 Topics: List Methods
- append()
- insert()
- remove()
- pop()

## 🧠 What I Learned Today
- list can be changed because it  is mutable.
- append() adds a new item at the end of the list
- insert() adds a new item at a specific index position
- remove() removes an item using value
- pop() removes an item using index
- pop() without index removes the last item
- pop() also returns the deleted value
- remove() removes only the first matching value
